#### code explanation Ajax has an object for a parameter. Besides the url key/value there is a success function and an error function. $.ajax({ url: 'http://sitebuilt.net:3002/api/ismatch/?name='+username+'&email='+email, success: function(data, status, response){ callback(response); }, error: function (response){ callback(response); } .bind(this) On error the most information is in the first parameter (the third param has nothing) Object {readyState: 0, responseText: "", status: 0, statusText: "error"} On success the most information is in the third parameter `success: function(data, status, response)`.
